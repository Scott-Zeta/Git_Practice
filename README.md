## Git configuration
1. System: store in /etc/gitconfig or Git in program files(windows)
    git config --system
2. User: ~/.gitconfig or user/yourUserName/.gitconfig
    git config --global
3. Porjec: project/.git/config
    git config

user ```git config --list``` to check all the configuration
